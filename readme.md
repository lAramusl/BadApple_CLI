Badapple in your terminal for adhd reasons.

It expects you to find video and audio and put it in project folder.

Requirements:
OpenCV 4.6.0

Instalation:
1. Find video and audio of badapple
    * video needs to be in .mp4 format
    * audio needs to be in .wav format
2. Rename video and audio
    * your_video.mp4 -> bad_apple.mp4
    * your_audio.mp4 -> bad_apple.wav
3. Run install.sh
4. Enjoy
